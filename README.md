# DC Map

This project is a simple interactive map highlighting a curated selection of places in Washington, DC that are either notable landmarks or personally meaningful locations. It is designed as a lightweight exploration tool to visualize geographic points of interest.

Live site: [Visit here!](https://rushikeshay.github.io/Folium-Map/)

## Technologies Used

This project is built entirely using Python and leverages the following libraries:

1. Pandas – for data manipulation and structuring location data
2. Folium – for creating interactive maps and visualizations
3. Geopy (Nominatim) – for geocoding locations and retrieving geographic coordinates
4. Time / Exception Handling – for managing API rate limits and ensuring reliable geocoding

## Features
1. Interactive map visualization of selected locations
2. Geocoding of place names into latitude and longitude
3. Custom markers for points of interest across Washington, DC
